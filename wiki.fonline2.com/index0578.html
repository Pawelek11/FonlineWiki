Description

Containers are anything used to store items. A locker is a common type of container, in your Fixboy you can find recipes for various types of containers. They make lives of FOnline2 wastelanders much easier and cleaner.

Each containers has it own capacity. You can see how much capacity is taken by looking on "Space Taken" number that is displayed just below container image after clicking on it.

Managing

  • Craftable containers can be picked up from ground when empty by using science skill.
  • Certain craftable containers can be rotated when empty by using repair skill.
  • Containers cannot be dropped in public locations.
  • Craftable containers cannot be disassembled into resources starting from session 3.
