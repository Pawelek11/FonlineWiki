Description

 
Hidden Cave
A Cave Hideout in the mountains.
Price N/A
Seller Name & Location N/A
Base Size Medium
Amount of Containers 3
Parking Places 10
Other Notes Map Can only be acquired by finding it at dungeons or buying from other players.

This is the third Cave base map variant available in the Fonline2. This base map is exported from old server called 2238 when it was base for novice players. It is a base with 2 map. Big outside/surface map on which you can park up to 10 vehicles and very small inner part where you can find only 3 lockers.

I would call this base bad. Not only it doesn't have enough containers, it is also, maybe just for me, not nice enough. I find that this cave looks pretty boring and not as it supposed to be - pretty.
