Description

Miscellaneous are well random, this category contains mostly useless strange items kept for collective value or the ones important in quests, dungeons etc. We can divide miscellaneous items into several categories.

Note: Altough most of items can be fitted into several category they are only being displayed once in this page. It is done so becuase the page is allready big enough and there is no need to duplicate the entries. And this is the reason for example Red Glow Keycard is being displayed only in "Quest and Dungeons items" and not in "Items for NPC's Collectors" even though it could be categorised in both sections.
