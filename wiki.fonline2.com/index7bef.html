Description

Character reroll is a new feature that was recently added to FOnline2. For a small price determined by formular of EXP/10 caps you can modify all attributes of your character - you will be able to change SPECIAL, skill points, traits and perks - everything.

What's lost in Reroll?

During reroll procedure your character will lose:

  • All read books bonuses.
  • Quests rewards.
  • All read blueprints will be lost.
  • All tents made by rerolled character will be deleted.
  • All bases made by rerolled characters will lose the ability to add/remove players. (unless you have other char that can add/remove players set up at that bases.)

You will need to obtain new books and redo quests.

The procedure

Firstly gather needed amount of caps. Use ~reroll to check how much you need. Then put them in the inventory of your character and type ~reroll again, if you do it right you will have a "all clear" message. Make sure your character is being placed in private place where no other people can get access. Make also sure its not the tent of character that is going to be rerolled as all tents made by this character will be deleted in the process.

Your character is in safe place? Necessary amount of caps are in inventory? Good! Use the ~deleteself (password) to start. Log off the character. It will be deleted from the FOnline2 memory banks after complete logout (when it disappear from the game after 3 minutes). In hex previously occupied by deleted character there will be F2 Hintbook with amount of experience equal to the number that rerolled character had. There will be also an implant leftover if there was one installed on the rerolled char.

Create a new character, you can use old nick if you want to. Go to base where the F2 book is. Use it. Your character will be bounded to the book which means that no other character will be able to use it. Still it can be looted or stolen from you so it advised to use up the book as soon as its possible.

Only fresh made level 1 character are able to be bound with the experience book.

After each book usage you will get equivalent of 3 levels. The book won't let you proceed if you have unused perk or more than 25 undistributed skill points.

The Last Perk

Its important to notice that normally during character leveling you can preserve last perk till level 99. In case of leveling character using a hintbook you cannot hold perk to that level and you have to pick it at level 24. This might be problematic and should be taken into consideration when planning your character (this is especially important for character using Medic perk).
